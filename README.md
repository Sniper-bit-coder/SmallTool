## 🌟 项目简介
这是一个专为中考学生打造的政治学科背诵工具。不同于传统的阅读式背诵，本项目通过“**学习-模拟默写-关键词比对**”的闭环逻辑，帮助用户将长篇大论的政治考点拆解为核心关键词，实现从“看到”到“背出”的质变。

---

## 🎨 设计语言
本项目在视觉上追求**“学术严肃性”**与**“现代交互感”**的平衡：
- **靛蓝色系**：代表理性、深度与法律的权威。
- **卡片式布局**：将复杂的专题结构化，减少认知负荷。
- **沉浸式暗色模式**：保护视力，适合长时间背诵。

---

## 🚀 核心功能
- **📚 结构化题库**：内置 18 个专题，涵盖友谊、法治、宪法、创新等核心考点。
- **🧠 记忆交替模式**：
  - **学习阶段**：展示完整答案，引导阅读。
  - **默写阶段**：隐藏答案，强制提取记忆，支持实时输入比对。
- **🔍 智能搜索**：全局实时搜索，瞬间定位模糊考点。
- **📊 数字化成就**：统计累计背诵时长、默写字数、练习次数，让进步肉眼可见。
- **💾 存档云同步（手动）**：支持 JSON 代码导出与导入，进度永不丢失。
- **📱 响应式适配**：完美兼容手机端，随时随地开启复习。

---

## 🛠️ 技术栈
- **Frontend**: 原生 HTML5, CSS3 (CSS Variables), JavaScript (ES6)
- **Data**: JSON 格式静态数据驱动
- **Architecture**: 模块化脚本设计 (`app-core.js`, `app-ui.js`, `features.js`)
